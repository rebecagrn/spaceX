# SpaceX

## 📥 How to run this project

```bash
# Open your terminal and clone the project, just type:
$ git clone https://github.com/rebecagrn/spaceX.git

# Install the dependencies
$ npm install
or
$ yarn

# DEVMODE: To run browser sync ans sass compiler
$ gulp

```

- HTML5
- CSS3
- Bootstrap 4.5
